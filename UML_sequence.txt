@startuml

participant Main
participant Game
actor Dealer
actor Player

== Constructors ==
    
Main -> Game : Construct new game
Game -> Dealer : Construct dealer
Game -> Player : Construct one player

== Begin ==

Main -> Player : Bet (Round 0)
Main -> Dealer : Deal cards (Round 1)
Dealer -> Dealer : Check for Black Jacks

alt The player as a Black Jack

Dealer -> Dealer : Give 3/2 of bet to player

else Continue normal game



end


@enduml
